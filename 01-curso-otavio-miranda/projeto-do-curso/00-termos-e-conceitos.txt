# Principais Tipos de Dados

* INT - valores inteiros de -2.147.483.648 a 2.147.483.647
* INT UNSIGNED - valores inteiros de 0 a 4.294.967.295
* BIGINT - valores inteiros de -9.223.372.036.854.775.808 a 9.223.372.036.854.775.807
* VARCHAR(numero) - valor textual com limite igual ao (numero), até 255
* TEXT - valor textual não limitado




# Modificações

* NOT NULL - Coluna não aceita valores nulos
* AUTO_INCREMENT - Valores são dados referentes aos valores já adicionados à tabela
* NULL - Permite valores nulos


# CONSTRAINT (restrições)

* PRIMARY KEY - chave primária da tabela
* UNIQUE KEY - só pode existir um por tabela (e-mail, por ex)
